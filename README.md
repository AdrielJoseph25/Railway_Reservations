# 🚆 Railway Reservations  

A web-based railway reservation system for booking, canceling, and managing train tickets. Built to simulate real-world ticketing systems with user and admin functionalities.  

---

## ⚙️ Technologies  
- Flask (Python)  
- MySQL  
- HTML, CSS, JavaScript  

---

## ✨ Features  

Here’s what you can do with Railway Reservations:  

- **User Authentication:** Secure login and registration system  
- **Book Tickets:** Search trains and reserve seats  
- **Cancel Tickets:** Easily cancel bookings  
- **Seat Availability:** View real-time seat status  
- **Ticket History:** Track previous bookings  
- **Admin Panel:** Manage trains and system data  

---

## 🧠 The Process  

- Designed a full-stack web application using Flask for backend logic  
- Integrated MySQL for handling user data, bookings, and train details  
- Built a responsive frontend using HTML, CSS, and JavaScript  
- Focused on implementing real-world booking flow and data handling
